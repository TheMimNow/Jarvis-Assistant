## Run Instructions

#### Create a conda environment

```bash
conda create -n jarvisassistant python=3.11 -y
```

```bash
conda activate jarvisassistant
```


```bash
pip install -r requirements.txt
```

```bash
streamlit run app.py
```


## Run Instructions for External SSD 

### Create a conda environment for External SSD


```bash
conda create --prefix .\jarvisassistant python=3.11 -y

```

```bash
conda activate .\jarvisassistant
```


```bash
pip install -r requirements.txt
```

```bash
streamlit run app.py
```
